# Resume Screening AI Project

## 📌 Project Overview
The **Resume Screening AI** project is a simple machine learning application that automatically evaluates how well a resume matches a given job description.  
It uses **Natural Language Processing (NLP)** techniques to calculate a match percentage, helping recruiters speed up the shortlisting process.

This project is designed for **internship-level learning**, focusing on clear logic and real-world application rather than complex deep learning models.

---

## 🎯 Objectives
- Automate resume screening using AI
- Reduce manual effort in resume shortlisting
- Demonstrate practical use of NLP and Machine Learning
- Build a beginner-friendly AI project for internships

---

## 🛠️ Technologies Used
- **Python**
- **Flask** (Web framework)
- **Scikit-learn**
- **NLTK**
- **TF-IDF Vectorizer**
- **Cosine Similarity**
